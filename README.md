# sncf-objet-trouve

Bot twitter publiant les déclarations de decouvertes d'objets dans le réseau sncf.

Compte twitter : https://twitter.com/ObjetSncf

Il renseigne la gare, la date, le nom de l'objet ainsi que sa nature.
